    Welcome To My Blog

    Sep 18, 2023
    2 min read
    Blog

    This is the start of my journey on my way to blog.

    I am not a professional blogger in any way right now.

    Yes, I have started blogs in the past but right now, I am finally feeling as if I am speaking to the public to express my thoughts.

    I think a lot. I think this will be a good outlet for me.

    I have setup auto publish using git and hosted this on github. I use hugo to build this static website.

    I use Obsidian to edit the markdown files.

    This has been so good so far.

    I have looked for alternatives but now I will start focusing on the content rather than the tech behind it.

    After starting a lot of blogs, I have realized that I am not really good at sticking to writing the blog rather I get too involved behind building the basic blogs of the blog.

    This is something that I have always dealt with in many of things that I have done in the past.

    I truly think, this will change everything.

    Everytime, I feel the urge to change something about the blog, I can just add it here.

    This is the only way to move forward.

    The website looks too good on the phone man.

    Today it’s Ganesh Chaturthi, so a good day to start projects.
